# Star
使用scrapy框架爬取星座網頁<br>
<br>
取出各星座與相關運勢內容<br>
<br>
每個一小時自動爬取<br>
<br>
並把資料存入數據庫中<br>
<br>
starName = 星座名稱
starDay = 當天日期
all1 = 整體運勢
love = 愛情運勢
business1 = 事業學業運勢
money = 財運運勢
link = 圖片
